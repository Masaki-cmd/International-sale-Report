# International-sale-Report
# 国際販売レポート分析プロジェクト README

## 概要

本プロジェクトは、提供された国際販売レポート（`International sale Report.csv`）データに対し、データクレンジングから始まり、売上トレンド、顧客行動、商品特性に至るまで多角的な分析を実施しました。目的は、ビジネスの現状を深く理解し、収益向上に繋がる具体的な課題と解決策を特定することです。

## 分析の目的

*   国際販売データの品質評価と改善。
*   月別売上トレンドの把握と主要変動要因の特定。
*   主要顧客および主要商品の特定。
*   売上貢献度の高い商品（トップスタイル）の成功要因分析。
*   売上は低いが多様な商品（ロングテール商品）の潜在的価値評価。
*   売上ゼロ商品の検証と適切なデータ処理方針の策定。
*   分析結果に基づいた収益向上戦略の提案。

## データソース

*   `International sale Report.csv`: 顧客、日付、商品スタイル、販売数量、売上金額などの情報を含む国際販売データ。

## 分析プロセスと主な結果

### 1. データクレンジングと前処理

*   **データ型の変換:** `PCS`, `RATE`, `GROSS AMT`列を数値型に、`DATE`列を日付型に変換。これにより、分析可能な形式にデータを整形しました。
*   **欠損値処理:**
    *   数値型列の欠損値は中央値で補完 (`PCS`, `RATE`, `GROSS AMT`)。
    *   カテゴリ型列の欠損値は最頻値で補完 (`Months`, `CUSTOMER`, `Style`, `Size`)。
    *   `SKU`列の欠損値は'Unknown'で補完。
    *   これにより、分析対象データから全ての欠損値を解消しました。
*   **データ品質課題の発見:** `CUSTOMER`列に'Feb-22'のような日付形式の値が混入しているなど、入力エラーの存在が確認されました。

### 2. 売上トレンド分析

*   **月別売上推移:** 総売上金額は月によって大きく変動し、特に2021年9月と2022年3月に顕著なピークが見られました。これは特定の季節イベントやプロモーション活動が影響している可能性があります。

### 3. 顧客・商品特性分析

*   **顧客別売上ランキング:** 'MULBERRIES BOUTIQUE'が群を抜いて最も高い売上貢献度を示しました。上位数社が全体の売上の大部分を占める集中型の顧客構造が見られます。
*   **スタイル別売上ランキング:**
    *   `SHIPPING`が最も高い売上を記録しましたが、これは配送料などサービス費用である可能性が高いと判断し、製品分析からは除外しました。
    *   `SHIPPING`を除いた上位スタイルは、`SET268`, `J0277`, `J0242`, `J0012`, `SET252` などであり、これらが主要な売上ドライバーであることが明らかになりました。

### 4. 収益向上に向けた詳細分析

*   **トップスタイルの深掘り (`SET268`を例に):**
    *   **顧客層と購買頻度:** `SET268`は28のユニークな顧客によって購入され、特定の顧客がこのスタイルを繰り返し購入している傾向が見られました。
    *   **関連商品（同時購入）:** `SET268`と共に購入される頻度が高い商品として、`SHIPPING`の他に `JNE3555`, `CMB5`, `JNE3548` などが特定されました。これはクロスセル戦略の機会を示唆します。
*   **ロングテール商品群の特定:** 売上が低いが0ではない265のスタイルが「ロングテール」商品として識別されました。これらの商品群には、潜在的な需要があるものの、十分に開拓されていない機会が存在する可能性があります。
*   **売上ゼロのスタイルの検証:** `TAGS`, `LABEL CHARGE`, `SHIPPING CHARGES`など、7つのスタイルが売上ゼロであることが確認されました。これらの中には実際の製品ではない管理項目が含まれており、分析対象から適切に除外するか、再分類する必要があります。

## 課題と提案

**課題1: データ品質の維持・向上**
*   **問題:** データの入力エラー（例: `CUSTOMER`列の誤った値）、不適切なデータ型、非製品項目の混在が分析精度を低下させています。
*   **提案:** データ入力時のバリデーションルールの導入、非製品スタイルのカテゴリ化または分析からの除外、`DATE`列変換時の`format`指定によるロバスト化。

**課題2: トップスタイルの成長戦略**
*   **問題:** 主要な売上ドライバーであるトップスタイルの成功要因が不明瞭で、その成長を最大化する戦略が不足しています。
*   **提案:** 他の上位スタイル（`J0277`, `J0242`など）についても`SET268`と同様の詳細分析を実施し、顧客セグメント、購買行動、プロモーション履歴などを深掘り。成功要因を特定し、それを他の商品や新商品開発に応用します。

**課題3: ロングテール商品の活性化と在庫最適化**
*   **問題:** 多くのロングテール商品が潜在的な売上機会を失っているか、非効率な在庫としてコストを発生させている可能性があります。
*   **提案:** ロングテール商品の中から、市場ニーズや季節性に基づいて特定のターゲット市場に響く商品を特定し、ターゲットプロモーション、バンドル販売、価格戦略の見直しを実施。同時に、需要の見込みが極めて低い商品については、在庫調整や販売中止を検討し、リソースを最適化します。

**課題4: 売上ゼロ商品の適切な管理**
*   **問題:** 売上ゼロの項目が分析データに残ることで、不正確な商品ポートフォリオ分析やリソース配分判断に繋がる可能性があります。
*   **提案:** 売上ゼロの項目を製品/非製品に分類。非製品は分析から除外または別カテゴリに計上。製品については、販売不振の原因を詳細に調査し、プロモーション、価格、製品改善などの対策を講じるか、廃止を検討します。

## 期待される効果

*   **売上増大:** トップスタイルの持続的成長と、ロングテール商品からの新たな収益創出。
*   **コスト削減:** 非効率な在庫や管理コストの削減、マーケティングリソースの最適化。
*   **データに基づいた意思決定:** クリーンで信頼性の高いデータと深い分析に基づき、より効果的なビジネス戦略を策定可能。
